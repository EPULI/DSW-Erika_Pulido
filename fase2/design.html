ETIQUETAS BASICAS HTML

Las etiqueta de HTML, son fragmentos de codigos con los se buscan crear elementos HTML, en donde los elementos son la estructura basica de HTML y esto cuentan con dos propiedades que son atributos y contenidos.

LAS PRINCIPALES ETIQUETAS SON:

TITTLE: Esta es utilizada para los titulos requeriddos

BODY: Este hace referencia al cuerpo del la paguina.

P: Esta etiqueta se utiliza para referrirse que va a comenzar un parrafo

B: Esta etiqueta es utilñizada para refereise a que el texto va en negrilla

SUB:Esta etiqueta es utilizada para referirise a los subtitulos que encontramos en el documentos HTML>

SELECT: Esta nos permite crear una lista de selección dento del documento HTML

TABLE: Esta etiqueta nos permite crear tablas dentro del docuemnto HTML

IMG:Con esta podemos ingresar imagenes al docuemnto HTML

A: Con esta etiqueta podemsos ingresar vinculos al documento HTML

SPAN: Esta etiqueta nos permite darle una sección al documeton dentro del HTML

ATRIBUTOS Y VALORES

Los atributos es la informacion adicional que se coloca con la etiqueta de apertura , esto se hace con el fin de dar a conocer o complementar informacón el archivo HTML.

los principales atributoa son:

ID: Este se utiliza con el fin de identicar un elemento unico

CLASS: Este es un identificador multiple

ALING: Este atributo es para darle alineacion al formato HTML

BORDER: Es utilizado para el borde del contenido en el HTML

BACKGROUND - COLOR: Este se utiliza para definir el colorde fondo del HTML

HREF: Este atributo se utiliza para ingresar un link o enlaces de un archivo HTML

HEIGHT: Con este atributo se define la altura del archivo HTML

WIDTH: Este es utilizado para el ancho de html

SRC: Este es para las imagenes

VALORES :

Los valores es la especificación de los atributos en donde se basa en una cantidad exacta o lamedida

en la que se utilizara el atributo y este va entre comillas despues del =.

EJEMPLOS DE ATRIBUTOS CON SU VALOR

ID: Este atributo puede tomar valores que identifiuen valores

ALGIN: Esta asume valores de alineacion como center, right, left, justify.

BORDER STYLE: El stilo del brde puede toamr valores como solid, dotted, dashed, double.

BG COLOR: Este es el color del fondo del HTML, en donde se puede utilizar cualquier color

HEREF: Esto es para adicionar el link o carpeta de dentro del HTML
